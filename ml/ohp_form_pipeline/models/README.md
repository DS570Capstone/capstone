# Models

This directory contains pre-downloaded model assets. They are gitignored due to size.

## pose_landmarker_heavy.task (30 MB)

Required for pose estimation. Download with:

```bash
curl -L -o models/pose_landmarker_heavy.task \
  https://storage.googleapis.com/mediapipe-models/pose_landmarker/pose_landmarker_heavy/float16/latest/pose_landmarker_heavy.task
```

Run from `ml/ohp_form_pipeline/`.
